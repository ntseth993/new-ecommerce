# ShopNow - Modern Ecommerce Store

  A full-stack ecommerce website built with React, Vite, Tailwind CSS, Express, and PostgreSQL.

  ## Features
  - Product browsing with search, filter, and sort
  - Product detail pages with ratings and related products  
  - Shopping cart with quantity management
  - Checkout and order placement
  - Order history

  ## Tech Stack
  - **Frontend**: React + Vite + Tailwind CSS + shadcn/ui
  - **Backend**: Express 5 + TypeScript
  - **Database**: PostgreSQL + Drizzle ORM
  - **API**: OpenAPI spec + Orval codegen
  - **State**: TanStack React Query
